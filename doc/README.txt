Directories
************
a, www - contain htdocs specific to a node (a), or mgmt (www)

bin - tools, binaries, etc that we do not modify

lookup - data for lookups

nginx - nginx binary

nginx_src - NGINX and modules src

skak - credentials

all - NGINX configurations common to all

log - where logs are written

py - python code we write. Right now it's just scripts (run from CLI), but 
     will be integrated into the web application (Django, I guess)

sh - shell scripts we write

lua - Lua scripts, see also http://wiki.nginx.org/HttpLuaModule. 

virtupy - virtual Python installation for third-party Python packages
