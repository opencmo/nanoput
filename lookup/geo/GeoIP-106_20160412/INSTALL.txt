Installing the GeoIP binary database
====================================

Simply copy the GeoIP-106_{date}.dat file to where ever the GeoIP.dat
file on your system is located.  For example on linux, type

% locate GeoIP.dat
/usr/local/share/GeoIP/GeoIP.dat
% cp GeoIP-106_20020701.dat /usr/local/share/GeoIP/GeoIP.dat

APIs to access the GeoIP.dat file are available here:
http://www.maxmind.com/app/api
